# ls
### **基本用法**
|命令|说明|
|---|---|
|`ls`|列出当前目录下内容|
|`ls /path/`|列出指定目录内容|

### **常用选项**
| 选项 | 说明 |
|------|------|
| `-l`| 以长格式显示（详细信息，包括权限、所有者、大小等） |
|`-d */`|仅列出目录|
| `-a`| 显示所有文件（包括隐藏文件，以 `.` 开头的文件） |
| `-h`| 与 `-l` 一起使用，以人类可读格式显示文件大小（如 KB、MB） |
| `-t`| 按修改时间排序（最新修改的在前） |
| `-r`| 反向排序 |
| `-S`| 按文件大小排序（大的在前） |
| `-R`| 递归列出子目录内容 |
| `--color` | 彩色显示（通常默认启用） |
|`--ignore="*.cpp"`|忽略某些文件,如"*.cpp"是忽略后缀为cpp的文件|
|`--time-style="+%Y-%m-%d"`|自定义时间格式|
