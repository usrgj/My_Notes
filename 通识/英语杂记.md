# 英语杂记

翻译上，英文是多名词，呈现一个静态的画面

作文
三段法
第一段，两句，背景，观点
第二段第一句要写得最漂亮
第三段结尾两三句，用新表达说旧观点

C = s + V
Independent clause (IC)
Dependent clause (DC)

pattern
1. IC,simple sentence
2. IC,IC,  compround sentence
3. IC,DC; DC,IC, complex snetence
4. DC,IC,DC;  IC,IC,DC ,  compround-complex
